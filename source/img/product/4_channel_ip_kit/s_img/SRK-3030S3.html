Overview

Kit Includes:

• SRN-473S PoE NVR with 1TB Hard Drive
• SND-L6013R Cameras x 2
• SNF-8010 5MP Fisheye Camera
• Remote Control, Mouse
• User Manual, Quick Guide, CD
• Power cable, Warranty Card
• Ethernet Cables (60ft) x 3

Key Features:

NVR (SRN-473S : 1ea)
• 4CH up to 8MP cameras supported
• 32Mbps network camera recording
• Plug & play by 4 PoE/PoE+ ports
• 1HDD - max 4TB
• HDMI / VGA local monitor
• Supports 2-way audio and PTZ control
• Supports protocols: Samsung & ONVIF Profile S
• POS Integration with text search
• Total 50W PoE budget

Camera (SND-L6013R : 2ea)
• Max. 2M (1920 x 1080) resolution
• 3.6mm fixed lens
• 0.15Lux@F1.8 (Color), 0Lux@F1.8 (B/W : IR LED on)
• 30fps@all resolutions (H.264)
• H.264, MJPEG dual codec, Multiple streaming
• MD, Tampering, micro SD/SDHC memory slot, PoE
• IR viewable length 15m
• Hallway view support (Rotate 90˚/270˚)
• LDC support (Lens Distortion Correction)

Camera (SNF-8010 : 1ea)
• Max. 5MP (1920 x 1080) resolution
• 360° Dewarp fisheye, 6 viewing modes
• 20fps@all resolutions (H.264)
• H.264, MJPEG dual codec, multiple streaming
• Motion detection, tampering, audio detection
• Micro SD/SDHC/SDXC memory slot, PoE
• Digital PTZ
