Key Features:

※ For running the app, Samsung Security Manager should be installed in your PC.
※ DDNS registration is needed in case of external network environment.

• Local/Remote connection to Samsung Security Manager
• View Layout/Device list by user
• View live streaming video with PTZ control
• Supported format : H.264/MJPEG
• Video Image Flip/Mirror, Video Image Capture
• Calendar Search/Playback/Bookmark functions
• Password function for security
• Quick & easy access at mobile environment & WIFI networks.
• Compatible with "Dynamic domain naming service" (DDNS)
• Multi-Language

This application uses FFmpeg and SDL under LGPL v2.1.
